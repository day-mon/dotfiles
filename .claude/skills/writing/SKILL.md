# Writing

Apply a consistent writing style to prose, then hand the draft back for review.

## Global rule (every style, always)
- **Never use em dashes (—).** Use a comma, colon, parentheses, or rewrite the sentence.

## Pick the style
- If the user named one (`/writing casual` or `/writing professional`), use it.
- Otherwise infer from context: chat/Slack/DM/notes → **casual**; docs/PRs/emails/anything
  work- or external-facing → **professional**. When genuinely unclear, ask before writing.

## Then
1. Read the matching style file in this skill directory: `casual.md` or `professional.md`.
2. Write the draft to match it.
3. Hand it back for review — it's a draft, not final.
